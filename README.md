# edmarbarros.com

Personal website for Edmar Barros — built with Astro, hosted on GitHub Pages.

## Development

```bash
pnpm install
pnpm dev
```

Site runs at http://localhost:4321.

## Build

```bash
pnpm build
pnpm preview
```
